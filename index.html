CPIT 280 Human-Computer Interaction

This course is an introduction to Human-Computer Interaction (HCI), where we discuss topics and issues related to designing effective interactive technologies that bring joy to users and improve the way they work and interact with technologies. Students in this class will learn concepts and techniques for prototyping, designing, and evaluating user interfaces.
